# Alien-Invasion
a 2D game in which the aim is to shoot down a fleet of aliens as they drop down the screen in levels that increase in speed and difficulty.

## Pygame Installation
```
python3 -m pip install -U pygame --user
```

## Run the game
```
python3 alien_invasion.py
```
